  • M t:
    • B d liu bao gm 768 mu t ph n thuc dn tc Pima Indian, vi 8 c im c o lng.
    • Mc tiu l d on liu bnh nhn c mc bnh tiu ng hay khng da trn cc c im y t.
  • D liu vo:
    • Pregnancies (S ln mang thai)
    • Glucose (Nng glucose huyt tng sau 2 gi)
    • BloodPressure (Huyt p tm trng, mm Hg)

    • Insulin (Insulin huyt thanh sau 2 gi, mu U/ml)
    • BMI (Ch s khi c th, kg/m)
    • DiabetesPedigreeFunction (Hm ph h tiu ng)
    • Age (Tui)
  • Kt qu: Outcome (0: Khng mc tiu ng, 1: Mc tiu ng)

# Tnh Median v IQR cho tng bin
for col in columns[:-1]:
    median = df_dataset[col].median()
    iqr = df_dataset[col].quantile(0.75) - df_dataset[col].quantile(0.25)
    q1 = df_dataset[col].quantile(0.25)
    q3 = df_dataset[col].quantile(0.75)
    outliers = ((df_dataset[col] < q1 - 1.5 * iqr) | (df_dataset[col] > q3 + 1.5 * iqr)).sum()
    print(f'{col}: Median = {median:.2f}, IQR = {iqr:.2f}, Outliers = {outliers}')

Khng tiu ng:

  • Glucose: Phn b tp trung ch yu khong 80 125 tc l phn ln ngi khe mnh c mc ng huyt bnh thng.
  • Age: Nhm khng bnh tp trung tui tr (2025), cng ln tui cng t ngi khng mc (nguy c tng dn).
  • BMI: Vn c nhiu ngi BMI 3040 nhng Outcome = 0 tc l ngoi bo ph cn c yu t khc mi c kh nng bnh

Mc bnh tiu ng (tiu ng):

  • Glucose: Phn b lch sang bn phi, nhiu ngi c Glucose 100, thm ch cao hn 150200 cho thy ng huyt cao lin quan trc tip n tiu ng.
  • Age: Phn b u nhiu tui.
  • BMI
    • BMI < 25 (bnh thng hoc gy): a phn khng mc tiu ng.
    • BMI 2530 (tha cn): bt u thy nhm mc tng ln.
    • BMI 3040 (bo ph): s ca mc vt tri hn hn so vi nhm khng mc ngha l bo ph l yu t nguy c mnh.
  • Pregnancy: Ngi c t 612 ln mang thai c xu hng mc tiu ng nhiu hn so vi khng mc
